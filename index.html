Introduction

Memory is one of the most essential cognitive functions our brains perform. It helps us interpret the world, make decisions, and communicate with others. Memory can be categorized in various ways such as short-term vs long-term memory, with long-term memory further divided into Implicit and Explicit Memory.

According to Parkin, Reid & Russo (1990), implicit and explicit memory differ in processing and retrieval of information. Implicit memory involves unconscious tasks and automatic skills, while explicit memory requires deliberate recall.

Implicit Memory

Implicit memory, also known as automatic or unconscious memory, allows us to perform tasks without consciously thinking about them. It is built through repetition and practice.

Explicit Memory

Explicit memory, or declarative memory, requires conscious effort to recall facts, events, and experiences. It is subdivided into semantic and episodic memory.

Question 1

Frank is trying to learn the names of all the students in his English course. Because Frank is actively and consciously remembering and recalling the names of students, this is considered ______ memory.

  • A. Implicit
  • B. Explicit
  • C. Episodic
  • D. Procedural

Answer: B. Explicit

Question 2

Katie is visiting Amsterdam, where riding bicycles is a common form of transportation. Even though she hasn’t ridden in a long time, Katie hops on and starts riding without actively thinking. Katie’s memory for riding a bicycle is an example of a(n) ______ memory.

  • A. Implicit
  • B. Sensory
  • C. Semantic
  • D. Explicit

Answer: A. Implicit
